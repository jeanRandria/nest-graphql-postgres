## Description

GraphQL API starter using [Nest](https://github.com/nestjs/nest) framework, [TypeORM](https://github.com/typeorm/typeorm) and Postgres.

## Installation

```bash
$ yarn 
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
